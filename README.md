# 🚀 Automatización de Envíos de Contenido vía WhatsApp

Este proyecto es una herramienta desarrollada en **Python** diseñada para optimizar la comunicación entre el Community Manager y sus clientes. Permite automatizar el envío de artes, videos y reportes mensuales de forma eficiente y profesional.

## 🎯 Objetivo
Facilitar la entrega de material visual a pequeños negocios, asegurando que el contenido (posts, historias, logotipos) llegue directamente al WhatsApp del cliente con un mensaje personalizado.

## 🛠️ Tecnologías Utilizadas
* **Python 3.12**: Lenguaje base del proyecto.
* **PyWhatKit**: Para la integración con WhatsApp Web.
* **PyAutoGUI**: Para la automatización de clics y envíos.
* **OS & Pathlib**: Para el manejo inteligente de rutas de archivos.

## ⚙️ Funcionalidades
- [x] Envío automático de imágenes y videos.
- [x] Personalización de mensajes con formato de texto.
- [x] Control de errores con bloques `try-except`.
- [x] Gestión de esperas inteligentes para asegurar la carga del contenido.
